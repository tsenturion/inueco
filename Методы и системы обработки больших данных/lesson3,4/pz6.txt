При проведении Чемпионата Европы по футболу собирается квалификационная группа из нескольких команд. Каждая из этих команд играет с каждой два матча - один домашний, другой гостевой.

В этом задании вам нужно из таблицы teams собрать список матчей, которые будут проведены в квалификационной группе.

Таблица teams содержит информацию о командах

| id         | name          |
|------------|---------------|
| 1          | Барселона     |
| 2          | ...           |

Создайте запрос, который соберет будущие матчи команды. Команды играют два раза - на своем поле и в гостях. Команды отсортированы по первой и второй колонке по возрастанию.

Для команд Милан, Спартак, Зенит результат будет следующий:

| home_team | away_team |
| --------- | --------- |
| Зенит     | Милан     |
| Зенит     | Спартак   |
| Милан     | Зенит     |
| Милан     | Спартак   |
| Спартак   | Зенит     |
| Спартак   | Милан     |
home_team - команда играет на своем поле
away_team - команда играет на гостевом поле
Команда не может играть сама с собой, поэтому значения в колонках не должны совпадать.

CREATE TABLE teams (
    id bigint,
    name varchar(255)
);
